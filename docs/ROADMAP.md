## Contributing to Roadmap

### How to Contribute

1. **Vote on features**: GitHub reactions on issues
2. **Submit proposals**: RFC process for major features
3. **Build integrations**: Partner program
4. **Community plugins**: Open-source contributions

### RFC Process

For major feature proposals:

1. Create RFC document
2. Submit to [rfcs repo](https://github.com/louienemesh/ResonantGenesis-rfcs)
3. Community discussion period (2 weeks)
4. Core team review
5. Decision and implementation
